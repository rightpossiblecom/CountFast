# 🎮 CountFast

**Stage:** Idea
**Status:** Not Started
**Core Benefit:** Improves quick thinking & visual estimation

---

# 1️⃣ Game Vision (User Perspective)

**CountFast** is a fast-paced brain game where:

Players see objects appear briefly on screen →
They must quickly count →
Select the correct number before time runs out.

It should feel:

* Fast
* Clean
* Stressful (in a good way)
* Rewarding
* “One more round” addictive

Short sessions. High replay value.

---

# 2️⃣ Core Gameplay Experience

## 🔁 Core Loop

1. Round starts
2. Objects appear (1–3 seconds)
3. Objects disappear
4. Player chooses the number
5. Immediate feedback
6. Next round instantly

Game ends after:

* 3 wrong answers
  OR
* Time mode ends

Then show results screen.

---

# 3️⃣ Main Game Modes (Keep It Simple)

### ✅ Classic Mode

* 3 lives
* Gets harder each round
* Speed increases
* Object count increases

Best for casual players.

---

### ✅ Time Attack Mode

* 60 seconds
* Unlimited mistakes
* Score = correct answers

More intense. Competitive feel.

---

# 4️⃣ Home Screen UX

When app opens:

Clean minimal screen with:

* Game title: **CountFast**
* Play (Primary Button)
* Mode selection (Classic / Time Attack)
* High Score
* Best Streak
* Small “How to Play” link

Feel:

* Calm but energetic
* Brain-training vibe

No clutter.

---

# 5️⃣ During Gameplay UX

## Visual Behavior

* Objects pop into screen with small bounce animation
* Clean background (no distraction)
* Timer bar at top
* Lives shown as hearts
* Score shown clearly

Objects disappear smoothly before answer selection appears.

---

## Answer Selection UI

After objects disappear:

Large number buttons appear (e.g., 8, 9, 10, 11)

Buttons:

* Big
* Rounded
* Easy to tap
* Clear typography

Wrong answer:

* Screen flashes red briefly
* Small vibration

Correct answer:

* Green flash
* Pleasant sound
* Smooth transition

Fast transitions. No delay.

---

# 6️⃣ Difficulty Progression (User Experience)

Game gradually becomes harder by:

* Increasing number of objects
* Reducing display time
* Adding similar-looking objects
* Slight random positioning
* Sometimes mixing sizes

Player should feel:
"I almost got it 😤"

Not:
"This is impossible."

---

# 7️⃣ Game Over Screen

Show:

* Final Score
* Best Score
* Longest Streak
* Accuracy %
* Play Again (big button)
* Back to Home

If new record:
🎉 Show “New High Score!” animation

Encourage replay instantly.
