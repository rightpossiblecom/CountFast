# CountFast Landing Page

The official landing page for **CountFast**, the brain training app that sharpens your split-second decision-making through fast-paced object counting.

## About CountFast

CountFast helps you improve your mental agility by challenging you to count objects quickly and accurately. Whether you're playing in Classic mode or racing against the clock in Time Attack, every session helps build a faster, sharper mind.

## Development

This is a [Next.js](https://nextjs.org) project built with React, TypeScript, and Tailwind CSS.

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm/yarn/bun

### Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the landing page.
